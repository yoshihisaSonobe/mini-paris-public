# mini-paris (Public contact repository)

mini-paris is a web app to help visualize and understand train/transit status around Paris.

This repository (`mini-paris-public`) is maintained as a public contact/support entry point.

- Web: https://miniparis.net
- Help: https://miniparis.net/help
- Legal: https://miniparis.net/legal

## What you can do here
- Report bugs (layout issues, wrong info, broken links, device-specific problems, etc.)
- Request features (new views, better navigation, missing info, etc.)
- Ask for data fixes (typos, incorrect spot details, etc.)
- General inquiries

## Contact
### GitHub Issues
Please open an Issue for bug reports and feature requests.

Helpful info to include:
- What you tried / expected / what actually happened
- Device (iPhone/Android/PC), OS, browser
- Screenshots (optional)

## Privacy / Analytics
- Google Analytics is disabled.
- See the Legal page for details: https://miniparis.net/legal

## Disclaimer
This repository is for public support only and does not contain the main application source code.
